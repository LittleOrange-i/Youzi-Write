# 柚子写作 - 后端 (主进程)

## 项目简介
本项目是“柚子写作”软件的 Electron 主进程部分，负责窗口管理、本地文件 IO、系统集成等核心功能。

## 主要功能
- **窗口管理**: 创建和管理主窗口、工具窗口等。
- **本地存储**: 使用 `electron-store` 持久化用户设置（如编辑器样式、网格线条配置等）。
- **文件系统交互**: 读取和保存章节内容、书籍信息。
- **自定义字体加载**: 支持读取本地字体文件并提供给渲染进程。
- **IPC 通信**: 提供丰富的接口供渲染进程调用。

## 目录结构
- `src/main`: 主进程代码。
  - `index.js`: 主进程入口文件。
- `src/preload`: 预加载脚本。
  - `index.js`: 定义暴露给渲染进程的 API。

## 技术栈
- **Electron**: 跨平台桌面应用开发框架。
- **Node.js**: 后端运行时。
- **electron-store**: 简单的持久化数据存储。

## 数据存储
用户设置（包括网格线条配置）保存在 `editorSettings` 键下，通过 Electron Store 进行读取和持久化。
