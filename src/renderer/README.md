# 柚子写作 - 前端渲染进程 (Renderer) <!-- 前端渲染进程主文档 -->

## 🌟 项目简介 <!-- 项目简介部分 -->
本模块是“柚子写作”应用的前端部分，基于 Vue 3 和 Electron 构建，负责提供用户交互界面、编辑器核心功能以及可视化工具。 <!-- 模块功能描述 -->

## 🚀 技术栈 <!-- 技术栈说明 -->
- **核心框架**: Vue 3 (Composition API) <!-- 使用 Vue 3 组合式 API -->
- **状态管理**: Pinia <!-- 响应式全局状态管理 -->
- **UI 组件库**: Element Plus <!-- 桌面端 UI 组件库 -->
- **样式处理**: Tailwind CSS & SCSS <!-- 原子化 CSS 与 预处理器 -->
- **编辑器核心**: TipTap (ProseMirror) <!-- 可扩展的富文本编辑器 -->
- **数据可视化**: ECharts & relation-graph-vue3 <!-- 图表与关系图展示 -->

## 📂 目录结构 <!-- 目录结构解析 -->
- `src/components`: 存放业务组件，如编辑器、地图、书架等 <!-- 业务组件目录 -->
- `src/views`: 页面级视图组件 <!-- 页面视图目录 -->
- `src/stores`: 全局状态仓库 <!-- 状态管理目录 -->
- `src/composables`: 可复用的逻辑组合函数 <!-- 逻辑封装目录 -->
- `src/assets`: 图片、图标、全局样式等静态资源 <!-- 静态资源目录 -->
- `src/utils`: 通用工具函数 <!-- 工具类目录 -->

## 🛠️ 核心功能模块 <!-- 核心功能解析 -->
1. **智能编辑器**: 支持码字音效、对白高亮、分屏编辑、实时统计。 <!-- 编辑器功能 -->
2. **书架管理**: 支持书籍创建、编辑、导出导入、**作品分类与标签墙管理**。 <!-- 书架管理功能，包含新增的标签功能 -->
3. **地图设计**: 提供画布工具、形状绘制、资源库管理。 <!-- 地图工具 -->
4. **创作辅助**: 包含随机起名、事序图、人物谱、世界观字典等。 <!-- 辅助工具 -->
5. **专注模式**: 锁定写作环境，帮助作者高效产出。 <!-- 专注功能 -->

## 📦 构建与运行 <!-- 运行说明 -->
前端部分作为 Electron 的渲染进程运行，通常通过主项目的 `pnpm dev` 启动。 <!-- 启动方式 -->
