# 柚子写作 - 前端

## 项目简介
本项目是“柚子写作”软件的渲染进程部分，基于 Vue 3、Element Plus 和 TipTap 构建，提供沉浸式的写作体验。

## 主要功能
- **沉浸式编辑器**: 支持多种字体、字号、行高设置。
- **网格线条**: 提供多种网格线条背景，辅助排版。
- **AI 侧边栏**: 集成 AI 辅助写作功能。
- **章节管理**: 方便的章节和笔记切换。
- **专注模式**: 排除干扰，专心码字。

## 目录结构
- `src/renderer/src/components/Editor`: 编辑器核心组件。
  - `EditorMenubar.vue`: 编辑器顶部的格式菜单栏。
  - `EditorDrawerContent.vue`: 侧边抽屉内容，包含网格线条等设置。
  - `EditorPanel.vue`: 编辑器主面板，管理编辑器实例。
- `src/renderer/src/views`: 页面视图组件。
  - `Editor.vue`: 编辑器主页面容器。
- `src/renderer/src/stores`: 状态管理。
  - `editor.js`: 编辑器相关状态。

## 技术栈
- **Vue 3**: 渐进式 JavaScript 框架。
- **Pinia**: Vue 的状态管理库。
- **Element Plus**: 基于 Vue 3 的组件库。
- **TipTap**: 无头编辑器框架。
- **Sass**: CSS 预处理器。
